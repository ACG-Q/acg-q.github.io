graph TD

F[钻石继承]

class A:
    def h(self):print('a')
class B(A):
    def h(self):print('b')
class C(A):
    def h(self):print('c')
class D(B,C):
    def h(self):print('d')
d = D()
d.h()
AB(A)C(A)D(C,B)输出
有有有有D
有有有无C
有有无无B
有无无无A

记一次程序汉化过程

记一次软件汉化的关键过程

汉化目标：PowerToys(微软小工具)

第一步：Radialix软件汉化

直接汉化PowerToys.exe，FancyZonesEditor.exe(PowerToys\modules)，ImageResizer.exe(PowerToys\modules)

程序名称新键本地化方案>>目标设置原因汉化对象
PowerToys.exe创建本地化文件托盘菜单汉化
FancyZonesEditor.exe创建多语言本地化文件明明本地化(创建本地化文件)成功了的，修改文件名称就失效了程序界面汉化
ImageResizer.exe创建本地化文件其实也可以不汉化，因为软件自带多语言程序界面汉化

第二步：js汉化

直接使用文本编辑器进行修改bundle.js(PowerToys\settings-html\dist)

通过查找/替换进行汉化，不过只能汉化以部分(选项一【PowerToys General Settings(常规设置)】以及弹窗菜单)，选项一中剩下没汉化的需要其余的汉化成功后，就会被自动替换。
